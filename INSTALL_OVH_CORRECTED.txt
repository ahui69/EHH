KROK 1: UPLOAD FOLDERU full/ NA VPS
═══════════════════════════════════════════════════════════

Method A: SCP
-------------
scp -r /workspace/full ubuntu@your-vps-ip:~

Method B: Git
-------------
ssh ubuntu@your-vps-ip
git clone https://github.com/ahui69/aktywmrd.git
cd aktywmrd/full

Method C: Tar.gz
----------------
# Lokalnie:
cd /workspace
tar -czf full.tar.gz full/
scp full.tar.gz ubuntu@your-vps-ip:~

# Na VPS:
ssh ubuntu@your-vps-ip
tar -xzf full.tar.gz
cd full
