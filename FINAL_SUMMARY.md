## 🆘 TROUBLESHOOTING

### "ModuleNotFoundError: fastapi"
```bash
source .venv/bin/activate
pip install -r requirements.txt
```

### "LLM_API_KEY not set"
```bash
nano .env
# Dodaj: LLM_API_KEY=twój_klucz
```

### "Port 8080 in use"
```bash
pkill -f uvicorn
# lub zmień port:
python -m uvicorn app:app --port 8081
```
