## Migration Guide

### From 1.0.0 to 2.0.0

1. **Update Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Update Configuration**
   ```bash
   cp .env.example .env
   # Update your configuration
   ```

3. **Update Database Schema**
   ```bash
   python migrate.py
   ```

4. **Update Frontend**
   ```bash
   # Replace old HTML files with new ones
   cp index_professional.html index.html
   ```

5. **Test Installation**
   ```bash
   python test_api.py
   ```
